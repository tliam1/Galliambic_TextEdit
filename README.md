# Galliambic_TextEdit
 A Text editor created using tkinter api

# Current Progress
- Basic keyword coloring
- Auto-Complete actions for quotations, ticks, single-quotes, parenthesis, and brackets
- Auto Indentation
- Text fonts
- Line Counter and Info Bar
